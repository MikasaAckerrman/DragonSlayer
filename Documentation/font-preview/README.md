# Font preview - what gets rendered with the embedded font

These three PNGs show **exactly** how the menu looks with the font
compiled into `libmenu.so`. The colours, bevels, tab merge, button
corners and grain are produced by the same code the engine runs
(`mainui_cpp/vgui1/src/Frame.cpp`, `TabPanel.cpp`, `Button.cpp`,
`TrackerScheme.cpp`); only the typeface is the input.

## What font is inside

`mainui_cpp/font/embedded_source/source.ttf` -> compiled into
`libmenu.so` via `font/embedded_font_data.cpp`. **Currently real
Microsoft Tahoma 1995-1999** so the menu looks identical to canon
CS 1.6 PC.

| Property | Value |
|----------|-------|
| Logical slot in TrackerScheme.res | `Tahoma` |
| Actual glyphs | Microsoft Tahoma Regular |
| Coverage | Latin 95/95, Russian 66/66 (А-Я а-я Ёё), Cyrillic block 122/255 |
| Size on disk | 310 752 bytes (~310 KB) |
| Embedded data array | ~1.9 MB autogenerated cpp -> ~310 KB in .so |

The slot name stays `Tahoma` so `TrackerScheme.res` references the
canon PC font name and no override is needed; the actual rendered
glyphs come from whatever TTF was last passed to `gen_embedded.py`.

## Legal

Microsoft Tahoma is proprietary Microsoft software. We embed it because
this build ships only via a **private repository**, used only by the
maintainer on devices where the maintainer holds a Microsoft licence
(Windows / Office) that grants Tahoma rights.

If you fork this work into a **public** repository or distribute the
APK to other people, swap `source.ttf` for a freely-licensed font
(DejaVu Sans Condensed, Noto Sans, etc.) and rerun
`gen_embedded.py` before publishing. See
`mainui_cpp/font/embedded_source/LICENSE` and `README.md` for details.

## Verifying the rendering matches the engine

`.audit/render_preview.py` uses the same RGB tuples as
`TrackerScheme.cpp` and the same bevel order as `Frame::paintBackground`.
If you spot a discrepancy between a preview here and the actual
on-device build, the bug is in the preview script - not the engine -
and the script should be fixed first.
